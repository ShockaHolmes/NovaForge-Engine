# Animation Studio

NovaForge Animation Studio supports two workflows.

## 1. Imported animation clips

Import an FBX, GLB, or GLTF file that contains animation clips. Select the imported model and the clips appear in the Imported Clips panel.

Click a clip to play it.

## 2. Created keyframe animations

1. Select an object.
2. Move, rotate, or scale it.
3. Set the keyframe time.
4. Press Capture Keyframe.
5. Move to another time and transform the object again.
6. Capture another keyframe.
7. Press Play Created.

The editor stores position, rotation, and scale keyframes.

## Keyboard shortcuts

- W: move
- E: rotate
- R: scale
- Delete: remove selected object
- Ctrl+D: duplicate selected object
